# @geophrase/react

The official React and Next.js hooks for integrating the Geophrase Connect address selector to reduce Return to Origin (RTO) costs.

👉 **[Read the full documentation and integration guide here](https://business.geophrase.com/docs)**

---

*Note: This package requires the [`@geophrase/core`](https://www.npmjs.com/package/@geophrase/core) peer dependency. If you are not using React, you should use the core package directly.*
