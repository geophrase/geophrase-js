# @geophrase/core

The core vanilla JavaScript library for Geophrase Connect. This package provides the foundational class and TypeScript definitions for integrating the Geophrase address selector to reduce Return to Origin (RTO) costs.

👉 **[Read the full documentation and integration guide here](https://business.geophrase.com/docs)**

---

*Note: If you are building a modern React or Next.js application, you should use the [`@geophrase/react`](https://www.npmjs.com/package/@geophrase/react) package instead, which wraps this core library in native hooks.*
